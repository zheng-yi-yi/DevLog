# FullStack Dev Log

[![VitePress](https://img.shields.io/badge/built%20with-VitePress-blue?style=flat-square)](https://vitepress.dev/)

A modern tech stack log and knowledge base covering Java, C#, Angular, Linux, and more.

## Overview

This project is a curated knowledge base for modern full-stack engineering:
- **Backend Mastery**: .NET Core, Java, Microservices.
- **Frontend Mastery**: Angular, TypeScript, State Management.
- **Infrastructure**: Linux, Docker, Kubernetes, CI/CD.

## Tech Stack

- **Documentation**: [VitePress](https://vitepress.dev/)
- **Configuration**: TypeScript-based config with separate `nav` and `sidebar` modules.
- **Content**: Multi-stack technical articles and Q&A.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run docs:dev
   ```

3. **Build Pre-view**:
   ```bash
   npm run docs:preview
   ```

## Project Structure

- `docs/` - Documentation source files.
  - `java/` - Java interview and knowledge base.
  - `index.md` - Homepage / Overview.
- `.vitepress/` - VitePress theme and system configuration.
- `package.json` - Build scripts and dependencies.
